AI 기능 상태

API Key 원문은 브라우저나 Supabase public table에 저장하지 마세요. 실제 운영에서는 Supabase Edge Function Secrets에 OPENAI_API_KEY, GOOGLE_API_KEY, ANTHROPIC_API_KEY 중 사용할 provider의 키를 저장하고, 이 화면은 등록 상태와 모델 설정만 관리합니다.
